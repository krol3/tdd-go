# BDD-go

Bdd using [Ginkgo](https://github.com/onsi/ginkgo)

### Setting project
`go mod init github.com/krol3/tdd-go.git``

### Setting Ginkgo
```
$ mkdir adder && cd adder
$ ginkgo bootstrap
```

Create new unit test: `ginkgo generate `
